# FerroTwin

Digital Twin platform for ferroalloy process monitoring.

## Repository Structure

- infra/          Azure Infrastructure as Code (Bicep)
- bootstrap/      Bootstrap scripts after infrastructure deployment
- dtdl/           Digital Twin Definition Language models
- functions/      Azure Functions
- inference/      Inference service
- dashboard/      Dashboard application
- docs/           Project documentation
- scripts/        Utility scripts

## Deployment

Infrastructure is deployed using Azure Bicep.
